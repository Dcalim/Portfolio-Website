Squared

June 2022 - July 2022

Summary

In this game, the goal is to reach a 1024 tile. The goal of the game is to Merge tiles of the same value by moving them up, down, left, or right. Merge successfully to create higher values, with the ultimate goal of reaching 1024. You lose the game if all tiles are filled and you can no longer make any moves. If you succeed or fail, the game prompts you to play again or return to the main menu.

The game is programmed using double for loops to iterate through each tile and if statements for correct tile merging. The random generated tiles are created by utilizing the random class for generating the values. This project was my final project in grade 11 for my Computer Science class and gave me fundamental hands on learning about the benefits of using user defined functions.
